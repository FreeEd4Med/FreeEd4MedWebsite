# ============================================================
# CAPTURE TIPS
# ============================================================
# 
# 1. Use a clean terminal with dark theme
# 2. Set terminal to 100 columns x 30 rows for consistency
# 3. Use: gnome-screenshot -a (for area selection)
#    or:  scrot -s filename.png
#    or:  flameshot gui
#
# 4. For video frames: 
#    ffmpeg -i video.mp4 -ss 00:00:05 -frames:v 1 frame.png
#
# 5. For composites, use GIMP or ImageMagick:
#    montage img1.png img2.png img3.png -tile 3x1 -geometry +5+5 composite.png
#
# 6. Optimize PNGs before upload:
#    optipng -o5 *.png
