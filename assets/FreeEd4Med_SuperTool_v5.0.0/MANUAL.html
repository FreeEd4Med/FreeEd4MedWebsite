1. Introduction & Setup

Welcome to the FreeEd4Med Media SuperTool v5.0. This comprehensive CLI (Command Line Interface) suite is designed for musicians, educators, and content creators to streamline media production, from audio mastering to social media distribution.

[IMAGE 1.1: Screenshot of the Main Menu v5.0]

Quick Start

  1. Open your terminal.
  2. Navigate to the Scripts folder.
  3. Run the tool: bash freeed_media_super_tool.sh

Prerequisites

  • OS: Linux (Ubuntu/Debian recommended) or WSL on Windows.
  • Dependencies: ffmpeg, ffprobe, python3, imagemagick.
  • Python Packages: Run pip install -r requirements.txt to install necessary libraries (like pysubs2, demucs, basic-pitch).

2. Creation Module & Songwriting Assistant

The Creation Module focuses on lyric videos, subtitle generation, and creative writing.

[IMAGE 2.1: Creation Module Sub-menu options]

Songwriting Assistant (New in v5.0)

Select Option 7 to launch the assistant. Features include:

  • Rhyme Finder: Finds perfect and near rhymes.
  • Thesaurus: Finds synonyms and related words.
  • AI Lyric Generator: Writes full lyrics based on your prompt (supports OpenAI and local Ollama).
[IMAGE 2.2: Screenshot of Songwriting Assistant]

Features

  • AI Whisper Generator: Automatically transcribe audio to .SRT subtitles using OpenAI's Whisper model. Supports auto-language detection.
  • Subtitle Composer: Launches the external subtitlecomposer GUI for fine-tuning timing.
  • Lyric Video (HardSub): Burns lyrics permanently into the video. Great for Instagram/TikTok where users can't toggle subs.
  • Lyric Video (SoftSub): Embeds subtitles as a track (mov_text). Viewers can turn them on/off.
  • Lyric Slideshow: Create a video from a folder of images + audio + subtitles.

4. Audio Lab

A complete suite for audio processing, conversion, and mastering analysis.

[IMAGE 4.1: Audio Lab Menu options]

Key Tools

  • Convert/Transcode: Batch convert to WAV, MP3, FLAC, AAC.
  • Loudness Normalize: Auto-level audio to -14 LUFS (Standard streaming target).
  • Vocal Cut: Create instrumentals by phase-canceling the center channel.
  • Mastering Report (Ardour Fixer):

    Runs a deep analysis using ardour_fixer.py. Checks LUFS, True Peak, LRA, and Phase correlation.

    • Presets: Spotify, YouTube, Apple Music, CD, Vinyl.
    • Features: Reference track comparison, Mid/Side export, Spectrum plotting.
[IMAGE 4.2: Example of a Mastering Report text output or PNG plot]

5. Notation Studio

AI-powered workflow to convert Audio into Sheet Music (PDF).

[IMAGE 5.1: Notation Studio workflow diagram (Audio -> Demucs -> Basic Pitch -> MuseScore)]

Workflow

  1. Separation: Uses demucs to isolate the instrument (e.g., piano stem) from a full mix.
  2. Transcription: Uses basic-pitch to convert the audio stem to MIDI.
  3. Notation: Uses MuseScore to render the MIDI into a readable PDF score.

7. Social Media Tools

The powerhouse module for distribution. Create content for all platforms in one go.

[IMAGE 7.1: Social Media Batch Generator Menu]

Batch Generator

Input one file (Video, Audio, or Image) and automatically generate optimized versions for:

  • TikTok: 1080x1920 (Vertical)
  • YouTube: 1920x1080 (Landscape)
  • Instagram: 1080x1080 (Square)
  • X (Twitter): Optimized Landscape
  • Facebook/Meta: 720p Optimized

AI Captions

Generate viral captions using AI backends (Ollama, OpenAI, Anthropic).

  • Tone: Casual, Professional, Funny, Inspiring.
  • Goal: Viral, Engagement, Promotion.
  • Output: Generates a JSON file with platform-specific captions and hashtags.
[IMAGE 7.2: Example of AI Generated Captions JSON output]

Uploads Hub

Directly upload to YouTube (OAuth) and TikTok (Dev API) from the terminal.
